# Miguel Sierra Alonso Portfolio

Portafolio profesional desarrollado con Next.js 15, TypeScript, Tailwind CSS y Framer Motion para presentar la trayectoria de Miguel Sierra Alonso como Squad Lead y Senior Software Engineer.

## Características

- Diseño moderno y responsive.
- Sección de perfil profesional y experiencia.
- Visualización de tecnologías y habilidades.
- Soporte para múltiples idiomas mediante rutas por locale.
- Optimizado para producción con renderizado estático y metadata SEO.

## Tecnologías utilizadas

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui-style components
- Vercel Analytics

## Estructura del proyecto

- app/: páginas y layout principal de la aplicación.
- components/: componentes reutilizables del portafolio.
- lib/: utilidades e internacionalización.
- assets/: recursos estáticos como imágenes.

## Requisitos previos

- Node.js 18 o superior
- npm o pnpm

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

La aplicación estará disponible en http://localhost:3000 (o en el puerto alternativo que indique Next.js si 3000 está ocupado).

## Construir para producción

```bash
npm run build
```

## Iniciar la versión de producción

```bash
npm run start
```

## Despliegue

Este proyecto está conectado a Vercel y realiza el despliegue automático en cada cambio publicado en la rama principal. La web está disponible en:

https://misial97-github-io.vercel.app
